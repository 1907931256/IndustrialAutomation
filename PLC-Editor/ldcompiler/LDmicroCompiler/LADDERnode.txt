LADDERdemo.zip note :

This project compiles with Microchip MPLAB C18

sdsOS stack is a demo stack, if you plan to develop with it, get latest version at :

http://sourceforge.net/projects/small-device-os/

blink.C is the file generated by the LDmicoCompiler (http://sourceforge.net/projects/ldcompiler/)
from blink.ld ladder file (http://www.cq.cx/ladder.pl)

user_LADDER.c is the main user file to use for you project.

this demo succesfull run on PicStar hardware : http://www.z-control.ch
