Project Information

About this project:

This is the SCADA ProcessViewer.

Crossplatform SCADA written on Free Pascal language (Lazarus IDE 0.9.28.2).

This project was registered on SourceForge.net on Nov 13, 2010.

This is an open source software, and distributed on the terms of GPL v.2.
